Question-1: What are the different ways to select an element in DOM?

The different ways to select an element in DOM are:

  1. 1.Using getElementById.
  2. 2.Using getElementsByClassName.
  3. 3.Using getElementsByTagName.
  4. 4.Using querySelector.
  5. 5.Using querySelectorAll.

Question-2: What are the difference between innerHTML,innerText and textContent?

  • 1.textContent: is all text contained by an element and all its children that are for purposes only
  • 2.innerText: returns all text contained by an element and all its child elements.
  • 3.innerHTML : returns all text,including html tags, that is contained by an element.

Question-3: What is event delegation in the DOM?

Event delegation refers to the process of using event propagation to handling events at a higher level in the DOM than the element on which the event originated.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM is a process that propagates events from an element to its parent elements.

Question-5:How do you create,add and remove elements using javaScript?

  • 1.Creating elements: document.createElement() method
  • 2.Adding elements:
    a.appendChild() method
    b.insertBefore() method
    c.Use append() and prepend() method
  • 3.Removing elements:
